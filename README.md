# Churrascometro

O objetivo do projeto é fornecer uma auternativa para quem deseja fazer um
churrasco entre amigos. O programa, baseado em algumas perguntas monta um
relatório personalizado e de fácil entendimento para que se possa comprar a
quantidade certa de insumos para o evento.

## Funcionalidades:

- Relatório de fácil compreensão e intuitivo
- Possibilidade de gerar o pdf para envio ao whatsapp e demais redes sociais



## instruções para execução
1) estando na pasta do projeto, execute o comando `npm install`
2) rode os comandos em sequência conforme abaixo, após os pacotes terem sido
instalados:
`npm run api`
`ng serve`
